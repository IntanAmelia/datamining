  2. Hitung jarak antara data uji dengan data latih

  3. Mengurutkan jarak dari yang terkecil hingga terbesar

  5. Memilih kelas mayor

    distance = 0
    for x in range(length):
        distance += np.square(data1[x] - data2[x])
       
    return np.sqrt(distance)

K-mean Clustering

Clustering adalah pengelompokan data kedalam beberapa cluster/kelompok berdasarkan kemiripan yang tinggi dan antar cluster memiliki kemiripan yang rendah atau bahkan berbeda.

Implementasi pada Clustering:

  • Segmenasi Citra (image processing)

  • Outlier Detection

  • CLustering Dokumen Berita

Jarak (Euclidian Distance)

Algoritma K-mean:

  1. Menentukan secara acak k titik data sebagai pusat cluster.

  2. Menandai masing-masing data masuk ke cluster tertentu dengan menghitung jarak ke masing-masing pusat cluster, yang memiliki jarak terdekat maka dimasukkan ke pusat cluster.

  3. Menentukan pusat cluster terbaru dengan menghitung rata-rata dari anggota cluster.

  4. Ulangi langkah 2 dan 3 sampai tidak ada anggota cluster yang berubah tempat.
